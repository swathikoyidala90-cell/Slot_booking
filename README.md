# Room Booking System

## Description
A simple Spring Boot application to manage room bookings with hourly rates and capacities.

## Installation
1. Clone the repository: 
   `git clone https://github.com/swathikoyidala90-cell/Slot_booking.git`
2. Navigate into the project: 
   `cd Slot_booking`
3. Build the project: 
   `./mvnw clean package`
4. Run the project: 
   `java -jar target/Slot_booking.jar`

## Usage
- Access the app at: `http://localhost:8080`
- Add, view, bookings and cancel room bookings.

## Dependencies
- Java 17
- Spring Boot
- PostgreSQL
